Proyecto UNO en Java

Descripción

Este proyecto es una recreación del famoso juego de cartas UNO, desarrollado en Java utilizando WindowBuilder para la interfaz gráfica. El juego permite a los usuarios interactuar con una versión digital de UNO, respetando las reglas clásicas del juego.

Tecnologías utilizadas

Java

WindowBuilder (para la interfaz gráfica)

Instalación y ejecución

Para ejecutar el proyecto, se recomienda usar Eclipse con las siguientes condiciones:

Tener instalado Java Development Kit (JDK) compatible con la versión utilizada en el proyecto.

Tener instalado el WindowBuilder en Eclipse para poder ejecutar y modificar la interfaz gráfica.

Importar el proyecto en Eclipse.

Ejecutar la clase principal del proyecto para iniciar el juego.

Uso

El juego se desarrolla mediante la interfaz gráfica generada por WindowBuilder.

Los jugadores pueden seguir las reglas del UNO clásico directamente en la aplicación.

No se requiere configuración adicional para comenzar a jugar.
